Auto hide navbar

A Javascript plugin to create transparent auto hide navbar

SEE FULL DEMO

Project Description:

A lightweight Javascript plugin to create transparent auto hide navbar. Navbar will automatically hide when the user scrolls down and will reappear when the user scrolls up. All it takes to implement auto hide navbar in your website is just a single line of javascript.
